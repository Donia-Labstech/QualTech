# QualTech — نظام رصد التربة والري الذكي

نظام متكامل (واجهة ويب) لمراقبة التربة (رطوبة، حرارة، ملوحة EC، حموضة pH، وعناصر NPK) والتحكم الآلي بصمام الري، مع لوحة تحكم، خريطة حقل، سجل وتقارير، محلّل ذكاء اصطناعي، وإمكانية تثبيت التطبيق (PWA).

**تطوير:** مختبر الأفكار الذكية — DONIA LABS TECH
**الموقع:** [Donialabstech.online](https://Donialabstech.online)

---

## ✨ الميزات

- لوحة تحكم حية بقراءات الرطوبة/الحرارة/EC/pH/NPK
- خوارزمية ري تلقائي (حدود رطوبة + حماية ملوحة) + تحكم يدوي
- خريطة حقل تفاعلية (Leaflet / OpenStreetMap)
- سجل تاريخي للقراءات + تصدير CSV
- محلّل ذكاء اصطناعي (محرك محلي + قابلية ربط نموذج خارجي مثل Claude عبر خادم وسيط)
- قابل للتثبيت كتطبيق (PWA) على الجوال والكمبيوتر
- كود جاهز للربط بجهاز استشعار حقيقي (ESP32 / Arduino) عبر API
- وضع طباعة نظيف للتقارير

## 🚀 التشغيل

### محلياً (للتجربة فقط)
افتح `index.html` مباشرة في المتصفح. سجّل الدخول بأي بيانات (نسخة تجريبية)، واستخدم سلايدرز "وضع المحاكاة" لتجربة كل السيناريوهات.

> ملاحظتان: ميزتا "تثبيت التطبيق" و"الاتصال بجهاز API حقيقي" لا تعملان عند الفتح المباشر (`file://`) — تتطلبان استضافة HTTPS حقيقية، مثل GitHub Pages بالأسفل.

### عبر GitHub Pages (مجاني — يفعّل HTTPS تلقائياً)
1. ادفع (Push) كل الملفات إلى هذا المستودع.
2. اذهب إلى **Settings → Pages**.
3. اختر **Branch: main** و **Folder: / (root)**، ثم Save.
4. بعد دقائق سيعمل الرابط الجذري مباشرة (لأن الملف الرئيسي مسمّى `index.html`): `https://<username>.github.io/<repo-name>/`

للتفاصيل الكاملة (ربط جهاز حقيقي، ربط نموذج ذكاء اصطناعي، ضبط الخوارزمية)، راجع [دليل التنفيذ](./qualtech-implementation-guide.md).
